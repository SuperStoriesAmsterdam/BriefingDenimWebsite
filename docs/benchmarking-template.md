## Monthly Review Checklist

Run this at the start of each monthly review with the team:

**Traffic**
- [ ] Which channel grew most this month?
- [ ] Which channel declined — and why?
- [ ] Are Dutch search terms ranking yet? (from Month 3)

**Leads**
- [ ] Which pipeline generated the most leads?
- [ ] Which pipeline has the worst abandonment rate?
- [ ] Are there leads sitting in a pipeline without a response? (escalation check)

**Conversion**
- [ ] What is the abandoned booking recovery rate? Is it improving?
- [ ] Which page has the highest exit rate before booking? (Clarity data)
- [ ] Is the concierge converting conversations to leads?

**Reputation**
- [ ] How many new Google reviews this month?
- [ ] Are negative reviews being responded to within 24 hours?
- [ ] TripAdvisor ranking movement?

**Content and SEO**
- [ ] Which pages gained search impressions this month?
- [ ] Are Dutch pages performing as expected?
- [ ] Any Clarity findings that require a page change?
